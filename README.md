# star-field


## Installation
`npm i @dvlpralamin/star-field`

## Basic Usage
```javascript


import React from 'react';
import ReactDOM from 'react-dom';
import StarField from "@dvlpralamin/star-field";

const App = () => {
  return (
    <StarField/>
  );
};

ReactDOM.render(<App />, document.getElementById('root'));
```
